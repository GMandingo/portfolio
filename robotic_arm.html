● Robotics & Control Systems

Control, Development and Teleoperation of WidowX 250s

In this project we were tasked with developing a ROS package to operate the WidowX 250s robot arm. the wx250s_teleop ROS package enabled remote control and automation for the 6-DOF robotic manipulator. The package supported multiple modes including Joint, Cartesian, and an automated Build mode for a "shrine" assembly task.

● ROS Framework

Project Learning and Experience

  • ● Developed practical experience with ROS, including building packages and working with nodes, topics, publishers, and subscribers.
  • ● Configured URDF models and used inverse kinematics to translate PS4 controller inputs into cartesian movement (x, y, z).
  • ● Developed a build mode to assemble a predetermined shrine, incorporating path planning and obstacle avoidance.
  • ● Integrated an RViz "digital twin" to monitor robot pose and coordinate transforms for during remote operation.
  • ● Enabled full operation from ps4 controller for all modes in order to enable both manual and remote user control.
